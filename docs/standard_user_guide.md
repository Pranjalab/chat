# Standard User Usage Guide

The Standard User role is optimized for simple Q&A based on approved documents.

## Access Level
- **Tools**: Access to basic logic and file reading tools.
- **Data Access**: **Read-Only** access to the `/data/user` directory.

## Usage Guidelines
- The agent can answer questions about existing files in the `user` folder.
- You cannot create new files or modify current documents.
- The environment is strictly sandboxed for maximum security.

## Common Task
"Search through the documents in the user folder and summarize the key findings about [Topic]."
